# Como Editar um Produto Existente
feature_id: edit_product
language: pt

O BiasharaAI permite editar os detalhes de qualquer produto registado no inventário. Pode atualizar o preço, o stock, o custo e outros dados a qualquer momento.

## Aceder ao Produto

1. No menu inferior, toque em **Inventário**.
2. A lista de produtos aparecerá no ecrã.
3. Localize o produto que deseja editar. Pode usar a barra de pesquisa no topo para encontrar o produto mais rapidamente.
4. Toque na linha do produto para abrir os detalhes.

## Editar os Dados

1. No ecrã de detalhes do produto, toque no botão **Editar** (ícone de lápis no canto superior direito).
2. O formulário de edição abrirá com os dados atuais já preenchidos.
3. Toque no campo que deseja alterar e faça a modificação.

## Campos que Pode Alterar

- **Nome do Produto**: Corrija ou atualize o nome.
- **Preço de Venda**: Atualize quando o preço ao cliente mudar.
- **Preço de Custo**: Atualize quando o fornecedor mudar o preço de compra.
- **Quantidade em Stock**: Ajuste manualmente se houver discrepância no stock.
- **Unidade de Medida**: Altere a unidade se necessário.
- **Código de Barras**: Adicione ou altere o código de barras do produto.
- **Categoria**: Mova o produto para outra categoria.
- **Imagem do Produto**: Substitua a foto do produto.

## Guardar as Alterações

1. Depois de fazer todas as alterações necessárias, toque em **Guardar**.
2. As alterações serão aplicadas imediatamente.
3. O histórico de transações anteriores não será afetado pelas alterações de preço.

## Ajuste de Stock

Se precisar ajustar o stock sem fazer uma venda ou compra, use a opção **Ajuste de Stock** disponível no ecrã de edição:

1. Toque em **Ajuste de Stock**.
2. Insira a quantidade atual real.
3. Adicione uma nota explicando o motivo do ajuste (por exemplo: "Inventário físico realizado").
4. Toque em **Confirmar Ajuste**.

Este ajuste ficará registado no histórico do produto para auditoria futura.

## Dicas

- Atualize o **Preço de Custo** sempre que receber mercadoria nova com preços diferentes. Isto garante que o cálculo de lucro seja preciso.
- Se o produto mudar de nome, verifique se o código de barras ainda está correto.
- Alterações de preço não afetam as vendas já registadas. Apenas as novas vendas usarão o preço atualizado.
- Use o campo de **notas** para registar informações adicionais sobre o produto, como fornecedor preferido ou prazo de validade médio.

## Erros Comuns

- **Preço de venda inferior ao custo**: O aplicativo irá alertá-lo, mas permite guardar mesmo assim. Verifique bem antes de confirmar.
- **Nome duplicado**: Se alterar o nome para um que já existe noutro produto, o aplicativo irá avisá-lo.

Todas as edições ficam registadas com data e hora para controlo interno.
