# Como Gerir Clientes
feature_id: customers
language: pt

O BiasharaAI permite registar e gerir os seus clientes. Pode guardar os dados de contacto, visualizar o histórico de compras e acompanhar dívidas pendentes.

## Aceder à Secção de Clientes

1. No menu inferior, toque em **Clientes**.
2. Verá a lista de todos os clientes registados.
3. Use a barra de pesquisa para encontrar um cliente rapidamente pelo nome ou número de telefone.

## Adicionar um Novo Cliente

1. Na secção **Clientes**, toque no botão **+** no canto superior direito.
2. Preencha os seguintes campos:
   - **Nome**: Nome completo ou nome de negócio do cliente.
   - **Telefone**: Número de telemóvel. Este campo é importante para contactar o cliente e para pagamentos via Mobile Money.
   - **Endereço** (opcional): Morada ou localização do cliente.
   - **Email** (opcional): Endereço de email para envio de recibos digitais.
   - **Notas** (opcional): Informações adicionais, como horário preferido ou tipo de produtos mais comprados.
3. Toque em **Guardar Cliente**.

## Ver Detalhes de um Cliente

1. Toque no nome do cliente na lista.
2. O ecrã de detalhes mostra:
   - **Informações de contacto** do cliente.
   - **Saldo de dívida atual**: Valor total em dívida não paga.
   - **Histórico de compras**: Lista de todas as transações realizadas pelo cliente.
   - **Histórico de pagamentos**: Datas e valores dos pagamentos de dívidas.

## Ver o Saldo de Dívida

No ecrã de detalhes do cliente:
- O **saldo de dívida** é exibido em destaque no topo.
- A cor vermelha indica dívida em aberto.
- A cor verde indica que não há dívidas pendentes.
- Toque em **Ver Dívidas** para ver o detalhe de cada dívida.

## Editar os Dados de um Cliente

1. Abra os detalhes do cliente.
2. Toque no ícone de **lápis** ou no botão **Editar**.
3. Faça as alterações necessárias.
4. Toque em **Guardar**.

## Eliminar um Cliente

1. Abra os detalhes do cliente.
2. Toque no menu de **três pontos** no canto superior direito.
3. Selecione **Eliminar Cliente**.
4. Confirme na janela de diálogo.

**Atenção**: Não é possível eliminar um cliente que tenha dívidas em aberto. Regularize as dívidas primeiro.

## Contactar um Cliente

Diretamente no ecrã de detalhes do cliente:
- Toque no número de telefone para fazer uma chamada.
- Toque no ícone de **WhatsApp** para enviar uma mensagem.
- Toque no ícone de **email** para enviar um email (se o email estiver registado).

## Associar um Cliente a uma Venda

Durante uma venda no POS:
1. Toque em **Adicionar Cliente**.
2. Pesquise e selecione o cliente.
3. O nome do cliente ficará associado à venda e aparecerá no recibo.

## Dicas

- Registe sempre o número de telefone do cliente. É essencial para gerir dívidas e enviar lembretes de pagamento.
- Um cliente bem registado facilita o acompanhamento de créditos e a fidelização.
- Pode exportar a lista de clientes para uma folha de cálculo a partir do menu de opções.
