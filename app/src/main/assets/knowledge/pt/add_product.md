# Como Adicionar um Novo Produto
feature_id: add_product
language: pt

Para adicionar um novo produto ao seu inventário, siga os passos abaixo.

## Aceder ao Inventário

1. Abra o aplicativo BiasharaAI.
2. No menu inferior, toque em **Inventário**.
3. Verá a lista de todos os produtos já registados.

## Adicionar um Novo Produto

1. Toque no botão **+** no canto superior direito do ecrã.
2. Será aberto o formulário de novo produto.

## Preencher os Dados do Produto

Preencha os seguintes campos:

- **Nome do Produto**: Digite o nome exato do produto. Use um nome claro e reconhecível. Por exemplo: "Arroz Branco 1kg" ou "Sabão em Pó 500g".
- **Preço de Venda**: Insira o preço pelo qual vai vender o produto ao cliente. Use apenas números. O símbolo da moeda é adicionado automaticamente.
- **Preço de Custo**: Insira o valor que pagou ao fornecedor por este produto. Este valor é usado para calcular o lucro.
- **Quantidade em Stock**: Insira a quantidade atual disponível para venda. Por exemplo: 50 unidades.
- **Unidade de Medida** (opcional): Escolha a unidade, como "unidade", "kg", "litro", "caixa".
- **Código de Barras** (opcional): Se o produto tiver código de barras, toque no ícone de câmara para escanear. Isto facilita as vendas no ponto de venda.
- **Categoria** (opcional): Associe o produto a uma categoria para facilitar a organização e pesquisa.

## Guardar o Produto

1. Depois de preencher todos os campos obrigatórios, toque em **Guardar** ou **Adicionar Produto**.
2. O produto será adicionado à lista de inventário imediatamente.
3. Receberá uma confirmação no ecrã.

## Dicas Importantes

- Preencha sempre o **Preço de Custo**. Sem este valor, o aplicativo não consegue calcular o lucro real do negócio.
- O **Stock** deve refletir a quantidade real disponível na loja. Mantenha o stock atualizado para evitar vender produtos esgotados.
- Se tiver muitos produtos, use **categorias** para organizar melhor o inventário.
- O nome do produto deve ser único. Se já existir um produto com o mesmo nome, o aplicativo irá avisá-lo.
- Pode adicionar uma **imagem** ao produto tocando no espaço da foto. Isto facilita a identificação rápida no ponto de venda.

## Erros Comuns

- **Campo obrigatório vazio**: O nome e o preço de venda são obrigatórios. O aplicativo não guarda sem eles.
- **Preço com vírgula ou ponto**: Use o formato correto da moeda local. O aplicativo aceita apenas números válidos.
- **Stock negativo**: Não é possível adicionar produtos com quantidade negativa.

Após guardar, o produto estará disponível no Inventário e também poderá ser adicionado a vendas no Ponto de Venda.
