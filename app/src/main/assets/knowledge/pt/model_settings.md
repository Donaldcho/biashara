# Como Gerir os Modelos de IA
feature_id: model_settings
language: pt

O BiasharaAI utiliza modelos de inteligência artificial para responder às suas perguntas, analisar dados e transcrever voz. Pode escolher entre modelos que funcionam diretamente no dispositivo ou modelos na nuvem.

## Aceder às Definições de Modelos de IA

1. Vá a **Definições** no menu principal.
2. Toque em **Modelos de IA** ou **Inteligência Artificial**.
3. O ecrã de gestão de modelos será exibido.

## Tipos de Modelos Disponíveis

### Modelos no Dispositivo (On-Device)

Estes modelos estão instalados diretamente no seu telemóvel:

- **Vantagens**:
  - Funcionam sem ligação à internet.
  - Os seus dados nunca saem do dispositivo.
  - Resposta rápida para tarefas simples.
  - Sem custo de dados.

- **Desvantagens**:
  - Requerem espaço de armazenamento no dispositivo (geralmente entre 500 MB e 4 GB).
  - Podem ser menos precisos em perguntas complexas.
  - Dispositivos mais antigos podem ser mais lentos.

### Modelos em Nuvem (Cloud)

Estes modelos processam as suas perguntas em servidores remotos:

- **Vantagens**:
  - Maior capacidade e precisão.
  - Atualizados regularmente.
  - Não ocupam espaço no dispositivo.

- **Desvantagens**:
  - Requerem ligação à internet.
  - Os dados são enviados para servidores externos.
  - Pode implicar custos de dados móveis.

## Ver os Modelos Instalados

No ecrã de Modelos de IA, verá uma lista de modelos com:
- **Nome do modelo**: Identificação do modelo.
- **Tipo**: No dispositivo ou Nuvem.
- **Tamanho**: Espaço que ocupa (para modelos locais).
- **Estado**: Instalado, A transferir, Disponível para instalar.
- **Uso**: Para que funções este modelo é usado (Chat, Voz, Agente, etc.).

## Selecionar o Modelo Ativo

1. Toque no modelo que deseja usar.
2. Toque em **Definir como Padrão** para o tipo de tarefa correspondente.
3. Pode ter modelos diferentes para diferentes funções:
   - **Modelo de chat**: Para responder às suas perguntas no Chat.
   - **Modelo de voz**: Para transcrição de áudio.
   - **Modelo do agente**: Para análise de negócio e sugestões.

## Descarregar um Novo Modelo

1. Na lista de modelos, toque num modelo com o estado **Disponível**.
2. Toque em **Instalar** ou **Descarregar**.
3. Aguarde o download concluir. O progresso será exibido.
4. Recomenda-se descarregar via Wi-Fi para evitar consumo excessivo de dados móveis.

## Eliminar um Modelo

Se precisar de libertar espaço de armazenamento:
1. Toque no modelo que deseja remover.
2. Toque em **Eliminar Modelo**.
3. Confirme a eliminação.
4. O modelo será removido do dispositivo mas poderá ser descarregado novamente quando necessário.

## Modelo Automático vs. Manual

Em **Definições de Modelos**, pode ativar o modo **Automático**:
- Com o modo automático ativo, o aplicativo escolhe automaticamente entre o modelo local e o modelo em nuvem consoante a disponibilidade de internet e a complexidade da pergunta.
- Com o modo **Manual**, define explicitamente qual modelo usar.

## Atualizar Modelos

Os modelos locais podem receber atualizações ocasionais:
1. Em **Definições de Modelos**, toque em **Verificar Atualizações**.
2. Se houver atualizações, toque em **Atualizar** para cada modelo.
3. As atualizações melhoram a precisão e corrigem erros.

## Dicas

- Se o seu dispositivo tem menos de 4 GB de RAM, use modelos compactos para melhor desempenho.
- Em zonas sem cobertura de dados, certifique-se de que tem um modelo local instalado.
- O modelo de voz local é suficiente para a maioria das situações do dia a dia.
- Se o chat parecer lento, verifique se há um modelo local mais leve disponível.
