# Gérer les modèles IA
feature_id: model_settings
language: fr

BiasharaAI utilise des modèles d'intelligence artificielle pour alimenter le Chat, l'Agent et la reconnaissance vocale. Vous pouvez choisir entre des modèles hors ligne (sur votre téléphone) et des modèles cloud (via Internet).

## Accéder aux paramètres des modèles

Allez dans **Paramètres → Modèles IA** ou **Paramètres → Intelligence artificielle**.

## Comprendre les types de modèles

**Modèles hors ligne (On-device)** :
- Fonctionnent sans connexion Internet.
- Vos données restent sur votre téléphone. Aucune information n'est envoyée à l'extérieur.
- Plus lents pour les analyses complexes.
- Nécessitent de l'espace de stockage sur votre téléphone (généralement 500 Mo à 4 Go selon le modèle).
- Idéal pour les boutiques dans des zones avec peu de connectivité.

**Modèles cloud** :
- Nécessitent une connexion Internet.
- Beaucoup plus puissants et précis.
- Analysent des données complexes plus rapidement.
- Peuvent impliquer des coûts selon votre abonnement.
- Vos données sont envoyées de façon chiffrée vers des serveurs sécurisés.

## Voir les modèles disponibles

Dans l'écran des paramètres de modèles, vous voyez la liste de tous les modèles disponibles. Pour chaque modèle :

- **Nom** : Le nom du modèle.
- **Type** : Hors ligne ou Cloud.
- **Taille** : L'espace de stockage requis (pour les modèles hors ligne).
- **Capacités** : Ce que le modèle peut faire (chat, voix, analyse, etc.).
- **Statut** : Installé, Disponible, ou Mise à jour disponible.

## Sélectionner un modèle

Appuyez sur le nom d'un modèle pour voir ses détails. Appuyez sur **Sélectionner comme modèle principal** pour l'utiliser par défaut. Vous pouvez aussi choisir des modèles différents pour des fonctions différentes :

- **Modèle de chat** : Utilisé pour le Chat IA.
- **Modèle de voix** : Utilisé pour la transcription vocale.
- **Modèle d'agent** : Utilisé pour les analyses de l'agent.

## Télécharger un modèle hors ligne

Si un modèle hors ligne n'est pas encore installé :

1. Appuyez sur le modèle dans la liste.
2. Appuyez sur **Télécharger** (une connexion Internet est nécessaire pour le téléchargement initial).
3. Attendez que le téléchargement soit terminé. Selon la taille, cela peut prendre quelques minutes.
4. Une fois téléchargé, le modèle fonctionne sans Internet.

## Mettre à jour un modèle

Quand une nouvelle version d'un modèle est disponible, une notification apparaît. Allez dans Paramètres → Modèles IA et appuyez sur **Mettre à jour** à côté du modèle concerné.

## Supprimer un modèle

Pour libérer de l'espace sur votre téléphone, supprimez les modèles que vous n'utilisez pas. Appuyez longuement sur le modèle et sélectionnez **Supprimer**. Vous pouvez le retélécharger plus tard si besoin.

## Mode de basculement automatique

Activez **Basculement automatique** pour que l'application utilise automatiquement le modèle cloud quand Internet est disponible, et le modèle hors ligne quand la connexion est absente. C'est le meilleur des deux mondes.

## Utilisation des données mobiles

Si vous utilisez des modèles cloud avec une connexion mobile, l'application consomme des données. Activez **Wi-Fi uniquement** pour n'utiliser les modèles cloud que lorsque vous êtes connecté au Wi-Fi.
