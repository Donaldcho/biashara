# Gérer les clients
feature_id: customers
language: fr

La section Clients vous permet d'enregistrer et de gérer vos clients réguliers. Cela est essentiel pour suivre les dettes, les achats fréquents et pour personnaliser votre service.

## Accéder à la section Clients

Appuyez sur l'onglet **Clients** dans la barre de navigation. La liste de tous vos clients enregistrés s'affiche.

## Ajouter un nouveau client

Appuyez sur le bouton **+** en bas à droite. Un formulaire s'ouvre avec les champs suivants :

**Nom complet** : Entrez le nom du client. C'est le seul champ obligatoire.

**Numéro de téléphone** : Entrez le numéro de téléphone. Il sera utilisé pour envoyer des reçus par SMS ou WhatsApp, et pour appeler le client en cas de besoin.

**Adresse** : L'adresse du client. Utile pour les livraisons.

**Note** : Un champ libre pour ajouter des informations supplémentaires. Par exemple : "Préfère payer le vendredi" ou "Client VIP".

Après avoir rempli les informations, appuyez sur **Enregistrer**.

## Voir le profil d'un client

Appuyez sur le nom d'un client dans la liste. Son profil s'ouvre avec :

- Ses informations de contact.
- Son **solde de dette** : le montant total qu'il doit encore payer.
- L'historique de ses achats.
- L'historique de ses paiements de dettes.

## Contacter un client

Depuis le profil du client, appuyez sur le numéro de téléphone pour appeler directement. Appuyez sur l'icône WhatsApp pour envoyer un message via WhatsApp (si disponible).

## Modifier les informations d'un client

Ouvrez le profil du client. Appuyez sur l'icône **Modifier** (crayon) en haut à droite. Mettez à jour les informations et appuyez sur **Enregistrer**.

## Supprimer un client

Pour supprimer un client, appuyez longuement sur son nom dans la liste, ou ouvrez son profil et utilisez le menu (trois points). Appuyez sur **Supprimer le client**.

Attention : vous ne pouvez supprimer un client que s'il n'a pas de dette en cours. Remboursez ou annulez d'abord toutes ses dettes.

## Rechercher un client

Utilisez la barre de recherche en haut de l'écran Clients. Tapez le nom ou le numéro de téléphone du client. Les résultats s'affichent instantanément.

## Utiliser les clients au POS

Lors d'une vente au POS, vous pouvez associer la vente à un client. Cela est obligatoire si le mode de paiement est **Dette**. Cela est optionnel pour les paiements en espèces ou Mobile Money, mais recommandé pour suivre l'historique d'achat de vos clients fidèles.

## Suivi du solde de dette

Le solde de dette de chaque client est visible directement dans la liste des clients. Un chiffre rouge indique qu'il a une dette en cours. Appuyez sur le client pour voir le détail et enregistrer un paiement.

## Clients fréquents et fidélité

Utilisez les notes et l'historique d'achat pour identifier vos meilleurs clients. Vous pouvez leur offrir des remises ou des avantages spéciaux pour les fidéliser.
