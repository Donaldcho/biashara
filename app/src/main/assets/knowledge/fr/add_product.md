# Ajouter un nouveau produit
feature_id: add_product
language: fr

Pour ajouter un nouveau produit à votre inventaire, suivez ces étapes simples.

## Ouvrir l'inventaire

Appuyez sur l'onglet **Inventaire** dans la barre de navigation en bas de l'écran. Vous verrez la liste de tous vos produits existants.

## Appuyer sur le bouton +

En bas à droite de l'écran, appuyez sur le bouton **+** (plus). Un formulaire d'ajout de produit s'ouvre.

## Remplir les informations du produit

Le formulaire contient plusieurs champs obligatoires et optionnels.

**Nom du produit** : Saisissez le nom du produit. Utilisez un nom clair et reconnaissable. Par exemple : "Farine de maïs 1 kg" ou "Savon Lifebuoy".

**Prix de vente** : Entrez le prix auquel vous vendez ce produit à vos clients. C'est le prix qui apparaîtra au point de vente.

**Prix de revient** : Entrez le prix auquel vous avez acheté ce produit. Ce champ est important pour calculer votre marge bénéficiaire. Si vous ne connaissez pas ce prix maintenant, vous pouvez le laisser vide et le compléter plus tard.

**Stock initial** : Indiquez la quantité de ce produit que vous avez actuellement en stock. Par exemple, si vous avez 50 sachets de farine, entrez 50.

**Unité** : Choisissez l'unité de mesure appropriée : pièce, kg, litre, paquet, carton, etc.

**Code-barres** : Si votre produit a un code-barres, vous pouvez l'entrer manuellement ou appuyer sur l'icône de scanner pour le lire automatiquement avec l'appareil photo.

**Catégorie** : Assignez une catégorie à votre produit pour faciliter la recherche. Par exemple : Alimentation, Boissons, Hygiène, Électronique.

**Stock minimum** : Indiquez la quantité minimale en dessous de laquelle l'application vous alertera. Si vous voulez être alerté quand il reste moins de 5 unités, entrez 5.

## Enregistrer le produit

Après avoir rempli tous les champs nécessaires, appuyez sur le bouton **Enregistrer** ou **Sauvegarder**. Le produit apparaît immédiatement dans votre liste d'inventaire.

## Conseils pratiques

- Ajoutez le prix de revient pour chaque produit. Cela permet à l'application de calculer votre profit exact.
- Utilisez des noms courts mais précis pour retrouver les produits rapidement au point de vente.
- Définissez des alertes de stock minimum pour ne jamais manquer d'un produit populaire.
- Si vous avez beaucoup de produits à ajouter, commencez par les plus vendus.

## Problèmes fréquents

Si le produit n'apparaît pas dans la liste après l'enregistrement, faites glisser l'écran vers le bas pour actualiser. Si le formulaire ne se ferme pas, vérifiez que tous les champs obligatoires sont remplis (ils sont indiqués par un astérisque).
