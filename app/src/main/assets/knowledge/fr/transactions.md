# Voir l'historique des transactions
feature_id: transactions
language: fr

L'historique des transactions vous donne un enregistrement complet de toutes les ventes, paiements et autres opérations effectuées dans votre boutique.

## Accéder à l'historique des transactions

Appuyez sur l'onglet **Transactions** dans la barre de navigation. La liste de toutes les transactions récentes s'affiche, avec les plus récentes en premier.

## Informations affichées

Pour chaque transaction, vous voyez :

- **Date et heure** : Quand la transaction a eu lieu.
- **Numéro de transaction** : Un identifiant unique.
- **Client** : Le nom du client si la transaction lui est associée.
- **Montant total** : Le montant de la vente.
- **Mode de paiement** : Espèces, Mobile Money ou Dette.
- **Statut** : Complété, Annulé, ou En attente.

## Voir le détail d'une transaction

Appuyez sur une transaction dans la liste. L'écran de détail s'ouvre et montre :

- La liste de tous les produits vendus avec leurs quantités et prix.
- Le sous-total, les taxes éventuelles et le total.
- Le mode de paiement et le montant reçu.
- La monnaie rendue (pour les paiements en espèces).
- Le nom de l'employé qui a effectué la vente (si plusieurs utilisateurs).

## Filtrer les transactions

Utilisez les filtres pour affiner la liste :

**Par date** : Appuyez sur l'icône de calendrier. Choisissez **Aujourd'hui**, **Cette semaine**, **Ce mois**, ou sélectionnez une plage de dates personnalisée. Appuyez sur **Appliquer**.

**Par type de paiement** : Filtrez pour voir uniquement les paiements en espèces, Mobile Money ou les ventes à crédit.

**Par client** : Entrez le nom d'un client dans la barre de recherche pour voir uniquement ses transactions.

**Par produit** : Recherchez un produit spécifique pour voir toutes les ventes qui le contiennent.

## Rechercher une transaction

Utilisez la barre de recherche en haut pour trouver une transaction par :
- Numéro de transaction.
- Nom du client.
- Montant exact.

## Imprimer ou partager un reçu

Depuis le détail d'une transaction, appuyez sur l'icône **Imprimer** pour imprimer un nouveau reçu sur votre imprimante Bluetooth. Appuyez sur l'icône **Partager** pour envoyer le reçu par SMS, WhatsApp ou email.

## Annuler une transaction

Si une vente a été enregistrée par erreur, vous pouvez l'annuler. Ouvrez la transaction et appuyez sur **Annuler la transaction**. Confirmez dans la boîte de dialogue.

Attention : l'annulation remet les produits en stock et annule la dette si applicable. Elle ne peut pas être faite si le paiement a déjà été déposé en banque.

## Exporter les transactions

Dans la section Rapports, vous pouvez exporter l'historique des transactions en format CSV pour l'analyser dans un tableur (Excel, Google Sheets). Cela est utile pour la comptabilité.

## Transactions récurrentes

Si vous avez des ventes régulières à des clients fixes (livraison hebdomadaire, etc.), l'historique vous montre la fréquence d'achat. Utilisez cela pour anticiper vos commandes fournisseurs.

## Sécurité

Toutes les transactions sont enregistrées avec l'horodatage exact. Elles ne peuvent pas être modifiées, seulement annulées. Cela garantit l'intégrité de vos données comptables.
