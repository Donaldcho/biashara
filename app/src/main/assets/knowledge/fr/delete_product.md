# Supprimer un produit
feature_id: delete_product
language: fr

Vous pouvez supprimer un produit de votre inventaire si vous ne le vendez plus. Cette action est irréversible, donc procédez avec soin.

## Accéder à l'inventaire

Appuyez sur l'onglet **Inventaire** dans la barre de navigation. La liste de vos produits s'affiche.

## Méthode 1 : Appui long

Appuyez longuement (maintenez le doigt appuyé pendant environ 2 secondes) sur la ligne du produit que vous souhaitez supprimer. Un menu contextuel apparaît avec plusieurs options :

- Modifier
- Dupliquer
- **Supprimer**

Appuyez sur **Supprimer**.

## Méthode 2 : Glisser vers la gauche ou la droite

Faites glisser la ligne du produit vers la gauche. Un bouton rouge **Supprimer** apparaît sur le côté droit. Appuyez sur ce bouton rouge pour lancer la suppression.

## Méthode 3 : Via la page de détail

Appuyez sur le produit pour ouvrir sa page de détail. Appuyez sur l'icône **Menu** (trois points verticaux) en haut à droite. Sélectionnez **Supprimer le produit**.

## Confirmer la suppression

Quelle que soit la méthode utilisée, une boîte de dialogue de confirmation apparaît. Elle vous demande :

**"Êtes-vous sûr de vouloir supprimer ce produit ?"**

- Appuyez sur **Supprimer** ou **Confirmer** pour valider la suppression.
- Appuyez sur **Annuler** pour revenir sans supprimer.

Cette confirmation existe pour éviter les suppressions accidentelles.

## Ce qui se passe après la suppression

Le produit disparaît immédiatement de votre liste d'inventaire. Il n'apparaîtra plus dans les résultats de recherche au point de vente.

Les transactions passées qui contenaient ce produit sont conservées dans l'historique. Les rapports anciens restent exacts.

## Quand supprimer un produit

Supprimez un produit quand :
- Vous ne le vendez plus du tout.
- Il a été ajouté par erreur avec un doublon.
- Le fournisseur a arrêté de le produire.

## Alternative à la suppression : désactiver un produit

Si vous n'êtes pas certain de vouloir supprimer définitivement un produit, vous pouvez le **désactiver** plutôt que le supprimer. Un produit désactivé n'apparaît plus au point de vente mais reste dans l'inventaire. Pour désactiver, ouvrez la page de détail du produit et désactivez l'interrupteur **Produit actif**.

Cela est utile pour les produits saisonniers que vous vendrez à nouveau plus tard.

## Que faire en cas d'erreur

Si vous supprimez un produit par erreur, vous devez le recréer manuellement. Il n'y a pas de fonction "annuler" pour les suppressions. C'est pourquoi la boîte de confirmation est importante.

Pour recréer le produit, appuyez sur le bouton **+** dans l'inventaire et renseignez à nouveau toutes les informations.
