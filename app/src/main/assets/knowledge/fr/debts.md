# Gérer les dettes clients
feature_id: debts
language: fr

La gestion des dettes vous permet de suivre les achats à crédit de vos clients. Vous savez à tout moment qui vous doit de l'argent et combien.

## Comment une dette est créée

Une dette est automatiquement créée quand vous effectuez une vente au POS et choisissez **Dette** comme mode de paiement. Le client prend la marchandise maintenant et paye plus tard.

Vous pouvez aussi créer une dette manuellement depuis la section Dettes.

## Accéder à la section Dettes

Appuyez sur l'onglet **Dettes** dans la barre de navigation. Vous voyez la liste de toutes les dettes en cours, classées par client.

Le total général des dettes en cours est affiché en haut de l'écran.

## Voir le détail d'une dette

Appuyez sur le nom d'un client dans la liste des dettes. Vous voyez :

- Le solde total dû par ce client.
- La liste de chaque transaction à crédit avec la date, les produits achetés et le montant.
- L'historique des paiements partiels déjà effectués.

## Enregistrer un paiement de dette

Quand un client vient payer sa dette (en tout ou en partie) :

1. Ouvrez la section **Dettes**.
2. Appuyez sur le nom du client.
3. Appuyez sur le bouton **Enregistrer un paiement**.
4. Entrez le montant payé par le client.
5. Choisissez le mode de paiement : Espèces ou Mobile Money.
6. Appuyez sur **Confirmer**.

Le solde restant est automatiquement mis à jour. Si le client a tout payé, la dette est marquée comme **Soldée** et disparaît de la liste des dettes actives.

## Marquer une dette comme payée manuellement

Si le client a payé d'une manière non tracée dans l'application (par exemple, il a payé à un employé qui a pris l'argent), vous pouvez marquer la dette comme payée manuellement. Ouvrez la dette, appuyez sur **Marquer comme payée**, et ajoutez une note explicative.

## Ajouter une dette manuellement

Pour créer une dette sans passer par le POS :

1. Appuyez sur **+** dans la section Dettes.
2. Sélectionnez le client.
3. Entrez le montant de la dette.
4. Ajoutez une description (exemple : "Achat de riz 5 kg le 15 mai").
5. Appuyez sur **Enregistrer**.

## Rappels de paiement

Vous pouvez envoyer un rappel de paiement à un client directement depuis son profil de dette. Appuyez sur l'icône d'envoi de message et choisissez SMS ou WhatsApp. Un message prérempli avec le montant dû est généré automatiquement. Vous pouvez le modifier avant d'envoyer.

## Filtrer et trier les dettes

Utilisez les options de filtre pour voir :
- Les dettes les plus anciennes (à récupérer en priorité).
- Les dettes les plus élevées.
- Les dettes d'un client spécifique.

## Rapport de dettes

Dans la section **Rapports**, vous pouvez voir un résumé de toutes vos dettes : total dû, évolution dans le temps, et clients avec le plus de dettes. Cela vous aide à gérer votre trésorerie.

## Conseils pratiques

- Contactez rapidement les clients dont la dette date de plus de 7 jours.
- Fixez une limite de crédit pour chaque client régulier pour éviter les grosses dettes.
- Ne laissez pas les petites dettes s'accumuler. Un rappel amical suffit souvent.
