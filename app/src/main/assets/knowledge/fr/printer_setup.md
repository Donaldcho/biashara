# Configurer l'imprimante Bluetooth
feature_id: printer_setup
language: fr

BiasharaAI prend en charge les imprimantes thermiques Bluetooth pour imprimer des reçus pour vos clients. La configuration est simple et ne prend que quelques minutes.

## Matériel requis

Vous avez besoin d'une imprimante thermique portable avec Bluetooth. Les marques courantes compatibles incluent : Xprinter, Rongta, EPSON, Star Micronics, Sewoo, et bien d'autres. Vérifiez que l'imprimante supporte le protocole ESC/POS.

## Étape 1 : Allumer l'imprimante

Mettez l'imprimante sous tension. Assurez-vous que le papier thermique est correctement installé. Le papier thermique a une face brillante (face imprimable) qui doit être tournée vers l'intérieur, vers la tête d'impression.

## Étape 2 : Activer le Bluetooth sur votre téléphone

Allez dans les **Paramètres** de votre téléphone Android. Activez le **Bluetooth**. Ne fermez pas les paramètres du téléphone tout de suite.

## Étape 3 : Appairer l'imprimante

Dans les paramètres Bluetooth de votre téléphone, appuyez sur **Rechercher des appareils** ou **Scanner**. Votre imprimante apparaît dans la liste des appareils disponibles. Son nom ressemble à "Printer_XXXX" ou "RPP02N" selon la marque.

Appuyez sur le nom de l'imprimante. Si un code PIN est demandé, entrez généralement **0000** ou **1234** (vérifiez le manuel de votre imprimante). L'appareil est maintenant appairé.

## Étape 4 : Connecter l'imprimante dans BiasharaAI

Ouvrez **BiasharaAI**. Allez dans **Paramètres → Imprimante**. Appuyez sur **Rechercher des imprimantes Bluetooth**.

La liste des imprimantes Bluetooth appairées s'affiche. Sélectionnez votre imprimante. Appuyez sur **Connecter**.

Un message de confirmation indique que l'imprimante est connectée.

## Étape 5 : Configurer le format du papier

Après la connexion, configurez le format :

- **Largeur du papier** : Choisissez 58 mm ou 80 mm selon votre rouleau de papier.
- **Densité d'impression** : Ajustez si les impressions sont trop claires ou trop sombres.

## Étape 6 : Imprimer un test

Appuyez sur **Imprimer un reçu de test**. L'imprimante doit imprimer un reçu de test avec le nom de votre boutique. Si l'impression est correcte, la configuration est terminée.

## Imprimer un reçu depuis le POS

Après une vente, une fenêtre propose d'imprimer le reçu. Appuyez sur **Imprimer**. Le reçu sort automatiquement de l'imprimante.

Vous pouvez aussi réimprimer un reçu depuis l'historique des transactions. Ouvrez la transaction et appuyez sur l'icône **Imprimer**.

## Problèmes fréquents

**L'imprimante n'apparaît pas dans la liste** : Vérifiez que le Bluetooth est activé sur le téléphone et que l'imprimante est allumée et en mode appairage.

**L'impression est illisible** : Vérifiez que le papier thermique est installé dans le bon sens (face brillante vers la tête d'impression).

**La connexion est perdue** : Allez dans Paramètres → Imprimante et reconnectez. Gardez l'imprimante proche du téléphone (moins de 10 mètres).

**Le texte est coupé** : Vérifiez que la largeur du papier configurée correspond à votre rouleau réel.
