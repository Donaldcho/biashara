# Utiliser les commandes vocales
feature_id: voice_commands
language: fr

Les commandes vocales vous permettent de contrôler BiasharaAI avec votre voix. C'est très pratique quand vos mains sont occupées ou quand vous servez plusieurs clients en même temps.

## Activer la saisie vocale

La saisie vocale est disponible dans deux endroits principaux :

**Dans le Chat IA** : Appuyez sur l'icône **Microphone** à côté du champ de texte en bas de l'écran Chat. Le microphone s'active.

**Dans le POS** : Appuyez sur l'icône **Microphone** dans la barre d'outils du point de vente pour dicter le nom d'un produit ou une commande.

## Parler clairement

Quand le microphone est activé, un indicateur visuel (cercle animé ou onde sonore) montre que l'application vous écoute. Parlez clairement et à une vitesse normale. Vous n'avez pas besoin de crier.

Tenez le téléphone à environ 20 à 30 centimètres de votre bouche. Si vous êtes dans un environnement bruyant, rapprochez le téléphone.

## Confirmer la transcription

Après que vous avez parlé, l'application transcrit ce que vous avez dit en texte. Ce texte s'affiche à l'écran avant d'être envoyé.

Vérifiez que la transcription est correcte. Si elle est correcte, appuyez sur **Envoyer** ou **Confirmer**. Si elle contient des erreurs, appuyez sur **Modifier** pour corriger le texte manuellement, ou appuyez sur **Recommencer** pour réessayer.

## Exemples de commandes vocales

**Pour poser des questions dans le Chat :**
- "Combien j'ai vendu aujourd'hui ?"
- "Qui me doit de l'argent ?"
- "Combien de sucre me reste-t-il ?"

**Pour chercher un produit au POS :**
- Dites simplement le nom du produit : "Farine de maïs" ou "Coca-Cola 500 ml"

**Pour des actions rapides :**
- "Ajouter un client" — l'assistant ouvre le formulaire d'ajout de client.
- "Voir mes ventes d'aujourd'hui" — l'assistant affiche le résumé du jour.

## Langues supportées

La saisie vocale fonctionne dans la langue configurée dans vos paramètres vocaux. Vous pouvez parler en français, en anglais, en swahili ou dans d'autres langues selon votre configuration. Consultez **Paramètres vocaux** pour changer la langue de reconnaissance vocale.

## Améliorer la reconnaissance vocale

Pour de meilleurs résultats :

- Parlez dans un endroit calme si possible.
- Prononcez les noms de produits clairement.
- Évitez de parler trop vite.
- Utilisez des phrases courtes et directes.
- Si votre accent pose problème, essayez d'articuler davantage.

## Que faire si la reconnaissance est mauvaise

Si l'application ne comprend pas bien ce que vous dites :

1. Vérifiez que le microphone de votre téléphone fonctionne bien.
2. Allez dans **Paramètres vocaux** et assurez-vous que la bonne langue est sélectionnée.
3. Essayez de parler plus lentement.
4. Nettoyez le microphone du téléphone si nécessaire.
5. En dernier recours, utilisez le clavier pour saisir votre texte.

## Confidentialité de la voix

L'audio de votre voix est traité localement sur votre téléphone si vous utilisez le modèle hors ligne. Il n'est pas envoyé sur Internet sauf si vous utilisez un modèle cloud.
