# Modifier un produit existant
feature_id: edit_product
language: fr

Vous pouvez modifier les informations d'un produit à tout moment depuis l'écran Inventaire.

## Accéder à l'inventaire

Appuyez sur l'onglet **Inventaire** dans la barre de navigation en bas. La liste de tous vos produits s'affiche.

## Trouver le produit à modifier

Faites défiler la liste pour trouver le produit. Vous pouvez aussi utiliser la barre de recherche en haut pour taper le nom du produit et le trouver rapidement.

## Ouvrir le formulaire de modification

Appuyez sur la ligne du produit que vous souhaitez modifier. L'écran de détail du produit s'ouvre. Appuyez ensuite sur l'icône **Modifier** (crayon) en haut à droite de l'écran.

Le formulaire de modification s'ouvre avec toutes les informations actuelles du produit déjà remplies.

## Modifier les informations

Vous pouvez modifier tous les champs du produit :

**Nom du produit** : Corrigez ou mettez à jour le nom si nécessaire.

**Prix de vente** : Mettez à jour le prix si vos tarifs ont changé. Le nouveau prix sera appliqué immédiatement aux prochaines ventes.

**Prix de revient** : Ajustez le prix d'achat si vous avez négocié de meilleures conditions avec votre fournisseur.

**Quantité en stock** : Vous pouvez corriger le stock manuellement si nécessaire. Notez que les ventes et réceptions de stock mettent à jour ce chiffre automatiquement.

**Code-barres** : Ajoutez ou corrigez le code-barres du produit.

**Catégorie** : Changez la catégorie si le produit a été mal classé.

**Stock minimum** : Ajustez le seuil d'alerte selon vos besoins.

**Unité de mesure** : Modifiez l'unité si vous vendez maintenant le produit différemment.

## Enregistrer les modifications

Après avoir fait vos changements, appuyez sur **Enregistrer**. Les nouvelles informations sont sauvegardées immédiatement.

Un message de confirmation apparaît brièvement en bas de l'écran pour confirmer que la mise à jour a réussi.

## Cas d'usage fréquents

**Changer le prix** : Si un fournisseur augmente ses prix, mettez à jour le prix de revient et ajustez votre prix de vente pour maintenir votre marge.

**Corriger le stock** : Après un inventaire physique, vous pouvez corriger les quantités qui ne correspondent pas à la réalité.

**Ajouter un code-barres** : Si vous avez ajouté un produit sans code-barres au début, vous pouvez l'ajouter maintenant pour accélérer les ventes au comptoir.

## Historique des modifications

L'application conserve un historique des changements de prix. Vous pouvez voir l'évolution des prix dans la section Rapports pour analyser vos marges dans le temps.

## Attention

Si vous modifiez le prix de vente d'un produit, ce changement n'affecte pas les transactions déjà enregistrées. Seules les nouvelles ventes utiliseront le nouveau prix.
