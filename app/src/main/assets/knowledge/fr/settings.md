# Paramètres de l'application
feature_id: settings
language: fr

L'écran Paramètres vous permet de personnaliser BiasharaAI selon vos besoins et le contexte de votre boutique.

## Accéder aux paramètres

Appuyez sur l'icône **Paramètres** (engrenage) dans la barre de navigation ou en haut à droite de l'écran d'accueil.

## Langue de l'application

Appuyez sur **Langue** pour choisir la langue de l'interface. Langues disponibles :
- Français
- English (Anglais)
- Kiswahili
- Et d'autres selon votre région.

Après avoir changé la langue, l'application redémarre automatiquement pour appliquer le changement.

## Devise

Appuyez sur **Devise** pour sélectionner la monnaie utilisée dans votre boutique. Exemples : Franc CFA (XOF, XAF), Shilling Kenyan (KES), Shilling Tanzanien (TZS), Franc Congolais (CDF), Naira Nigérian (NGN), etc.

Choisissez aussi le **format d'affichage des montants** : position du symbole de devise, séparateur de milliers, nombre de décimales.

## Taux de taxe (TVA)

Si votre boutique collecte des taxes sur les ventes, configurez le taux ici. Appuyez sur **Taux de taxe** et entrez le pourcentage applicable. Par exemple, entrez 18 pour 18 %.

Vous pouvez aussi choisir si le prix affiché au POS est TTC (toutes taxes comprises) ou HT (hors taxes). Activez **Taxes incluses dans les prix** si vos prix affichés incluent déjà la taxe.

Si vous ne collectez pas de taxes, laissez le taux à 0 %.

## Imprimante

Configurez votre imprimante Bluetooth ici. Appuyez sur **Imprimante** pour rechercher et connecter une imprimante thermique. Voir la section "Configurer l'imprimante" pour les instructions détaillées.

Vous pouvez aussi configurer :
- **Format du reçu** : Largeur du papier (58 mm ou 80 mm).
- **En-tête du reçu** : Nom de votre boutique, adresse, numéro de téléphone, slogan.
- **Pied de page** : Message de remerciement ou informations légales.
- **Logo** : Ajoutez le logo de votre boutique sur le reçu.

## Sauvegarde des données

Appuyez sur **Sauvegarde** pour configurer la protection de vos données.

**Sauvegarde locale** : Enregistre vos données sur la carte mémoire ou dans la mémoire du téléphone. Appuyez sur **Sauvegarder maintenant** pour créer une sauvegarde immédiate.

**Sauvegarde automatique** : Activez l'interrupteur pour que l'application sauvegarde automatiquement vos données chaque jour ou chaque semaine.

**Restaurer** : Si vous changez de téléphone ou après une réinstallation, appuyez sur **Restaurer** et sélectionnez le fichier de sauvegarde.

## Informations de la boutique

Entrez les informations de votre boutique :
- Nom de la boutique.
- Adresse.
- Numéro de téléphone.
- Email (optionnel).
- Numéro fiscal (si applicable).

Ces informations apparaissent sur les reçus imprimés.

## Notifications

Configurez les types de notifications que vous souhaitez recevoir : alertes de stock, rappels de dettes, rapports journaliers.

## Sécurité

Configurez un **code PIN** ou utilisez l'**empreinte digitale** pour protéger l'accès à l'application. Utile si vous partagez votre téléphone avec des employés.
