# Le grand livre comptable
feature_id: ledger
language: fr

Le grand livre est un registre de toutes les entrees et sorties d argent de votre boutique. Il vous donne une image complete de votre sante financiere et vous aide a comprendre votre solde courant.

## Acceder au grand livre

Appuyez sur l onglet **Livre** ou **Grand livre** dans la barre de navigation. La liste de toutes les ecritures s affiche par ordre chronologique, avec les plus recentes en premier.

## Comprendre le solde courant

En haut de l ecran, vous voyez le **Solde courant** : c est la difference entre toutes vos entrees d argent et toutes vos sorties d argent depuis le debut. Un solde positif signifie que votre boutique a plus recu que depense. Un solde negatif indique que vos depenses ont depasse vos recettes.

Le solde est mis a jour automatiquement apres chaque vente, paiement ou depense enregistree.

## Lire les ecritures

Chaque ligne du grand livre est une ecriture. Elle indique :

- **Date** : Quand l operation a eu lieu.
- **Description** : Ce que represente cette ecriture (exemple : Vente POS, Achat fournisseur, Paiement loyer).
- **Montant** : Le montant de l operation.
- **Type** : Entree (argent recu, affiche en vert) ou Sortie (argent depense, affiche en rouge).
- **Solde apres** : Le solde total apres cette ecriture.

## Types d ecritures automatiques

Le grand livre est alimente automatiquement par :

- Les **ventes au POS** en especes ou Mobile Money (entrees).
- Les **paiements de dettes** recus de clients (entrees).
- Les **depenses** enregistrees via le scan de recus ou manuellement (sorties).
- Les **achats fournisseurs** enregistres dans les entrees de stock (sorties).

## Ajouter une ecriture manuellement

Pour ajouter une ecriture qui n est pas liee a une vente ou une depense habituelle :

1. Appuyez sur le bouton **+** en bas a droite.
2. Choisissez le type : **Entree** ou **Sortie**.
3. Entrez le montant.
4. Entrez une description claire. Exemple : "Remboursement TVA", "Apport personnel", "Pret bancaire recu".
5. Choisissez la date (par defaut : aujourd hui).
6. Appuyez sur **Enregistrer**.

L ecriture apparait immediatement dans le grand livre et le solde courant est mis a jour.

## Exemples d ecritures a ajouter manuellement

- **Apport de fonds personnels** : Quand vous ajoutez de l argent personnel dans la caisse de la boutique.
- **Retrait personnel** : Quand vous prenez de l argent de la boutique pour vos besoins personnels.
- **Paiement de loyer** : Si le paiement du loyer n est pas enregistre autrement.
- **Remboursement de pret** : Paiement d une echeance de credit bancaire.
- **Recettes non liees aux ventes** : Commission, revente de materiel, etc.

## Filtrer les ecritures

Utilisez les filtres en haut de l ecran pour afficher :
- Seulement les entrees.
- Seulement les sorties.
- Les ecritures d une periode specifique.
- Les ecritures d une categorie specifique.

## Exporter le grand livre

Appuyez sur **Exporter** pour telecharger le grand livre en format CSV ou PDF. Partagez-le avec votre comptable pour faciliter la preparation de vos declarations fiscales.

## Modifier ou supprimer une ecriture

Pour modifier une ecriture manuelle, appuyez longuement dessus et selectionnez **Modifier**. Les ecritures automatiques (ventes, depenses) ne peuvent pas etre modifiees directement dans le grand livre. Corrigez-les depuis la section correspondante.

## Conseils

Verifiez votre grand livre chaque semaine. Assurez-vous que toutes les entrees et sorties d argent sont enregistrees. Un grand livre precis vous aide a prendre de meilleures decisions pour votre boutique.
