# Configurer les paramètres de l'agent IA
feature_id: agent_settings
language: fr

Les paramètres de l'agent vous permettent de contrôler comment l'intelligence artificielle surveille votre boutique et quand elle vous envoie des suggestions.

## Accéder aux paramètres de l'agent

Depuis l'onglet **Agent**, appuyez sur l'icône **Paramètres** (engrenage) en haut à droite de l'écran. Vous pouvez aussi y accéder via **Paramètres → Agent IA**.

## Activer ou désactiver l'agent

En haut des paramètres, vous trouverez un interrupteur principal **Activer l'agent**. Désactivez cet interrupteur pour arrêter toutes les analyses et suggestions de l'agent. Cela est utile si vous partez en vacances ou si vous voulez utiliser l'application de façon simple sans suggestions.

## Gérer les compétences (Skills)

L'agent IA fonctionne grâce à des compétences spécialisées. Chaque compétence couvre un domaine spécifique.

**Compétences disponibles :**

- **Surveillance des stocks** : Alerte quand un produit est presque épuisé.
- **Suivi des dettes** : Rappelle les dettes anciennes et les clients qui n'ont pas payé.
- **Analyse des ventes** : Détecte les tendances et les produits performants.
- **Surveillance des flux de trésorerie** : Alerte si vos dépenses dépassent vos revenus.
- **Analyse des marges** : Signale les produits avec des marges trop faibles.
- **Rappels de tâches** : Rappels pour les tâches régulières (comptage de caisse, rapport).

Pour activer ou désactiver une compétence, appuyez sur l'interrupteur à côté de son nom.

## Configurer les seuils d'alerte

Pour chaque compétence, vous pouvez ajuster les seuils qui déclenchent une alerte :

**Alerte de stock** : Définissez le nombre de jours de stock restant qui déclenche l'alerte. Par défaut : 3 jours.

**Alerte de dette** : Définissez le nombre de jours après lequel une dette non payée déclenche un rappel. Par défaut : 7 jours.

**Alerte de flux de trésorerie** : Définissez le solde minimum en dessous duquel vous êtes alerté.

Pour modifier un seuil, appuyez sur le chiffre à côté du paramètre et entrez votre valeur préférée.

## Planifier les analyses

Vous pouvez configurer quand l'agent effectue ses analyses :

**En temps réel** : L'agent analyse chaque transaction immédiatement. Recommandé pour la surveillance des stocks et des dettes.

**Quotidien** : L'agent génère un résumé quotidien à l'heure que vous choisissez. Recommandé pour les rapports de ventes.

**Hebdomadaire** : Un rapport complet de la semaine. Recommandé pour les analyses de tendances.

Pour configurer le planning, appuyez sur **Planification des analyses** et choisissez la fréquence et l'heure pour chaque type d'analyse.

## Notifications push

Activez les **Notifications push** pour recevoir des alertes sur votre téléphone même quand l'application est fermée. Vous pouvez choisir quels types d'alertes envoient des notifications (par exemple, seulement les alertes de stock critique).

## Langue des suggestions

L'agent génère ses suggestions dans la langue configurée dans les paramètres généraux de l'application. Consultez **Paramètres → Langue** pour changer la langue.

## Réinitialiser les paramètres de l'agent

Pour revenir aux paramètres par défaut, faites défiler jusqu'en bas et appuyez sur **Réinitialiser les paramètres de l'agent**. Confirmez dans la boîte de dialogue.
