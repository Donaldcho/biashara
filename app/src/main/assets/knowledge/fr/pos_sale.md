# Effectuer une vente au point de vente (POS)
feature_id: pos_sale
language: fr

Le point de vente (POS) est l'écran principal pour enregistrer les ventes à vos clients. Il est rapide et facile à utiliser.

## Ouvrir le point de vente

Appuyez sur l'onglet **POS** ou **Vente** dans la barre de navigation. L'écran de caisse s'ouvre.

## Ajouter des produits au panier

**Recherche par nom** : Tapez le nom du produit dans la barre de recherche. Les résultats apparaissent en dessous. Appuyez sur le produit pour l'ajouter au panier.

**Grille de produits** : Si vos produits les plus vendus sont affichés en grille, appuyez simplement sur l'image ou le nom du produit pour l'ajouter.

**Scanner un code-barres** : Appuyez sur l'icône de scanner et pointez l'appareil photo vers le code-barres du produit. Il s'ajoute automatiquement au panier.

## Gérer les quantités dans le panier

Une fois le produit dans le panier :

- Appuyez sur **+** pour augmenter la quantité.
- Appuyez sur **-** pour réduire la quantité.
- Appuyez directement sur le chiffre de la quantité pour saisir une valeur précise au clavier.
- Glissez la ligne vers la gauche pour retirer le produit du panier.

## Vérifier le total

Le total de la vente s'affiche en bas du panier. Il se met à jour automatiquement chaque fois que vous ajoutez ou retirez un produit.

## Choisir le mode de paiement

Appuyez sur le bouton **Facturer** ou **Encaisser**. Un écran de paiement s'affiche avec trois options :

**Espèces (Cash)** : Le client paie en argent liquide. Entrez le montant remis par le client. L'application calcule automatiquement la monnaie à rendre.

**Mobile Money** : Le client paie via un service mobile (M-Pesa, Orange Money, MTN Money, etc.). Sélectionnez le réseau et entrez le numéro de référence de la transaction si nécessaire.

**Dette (Crédit)** : Le client paie plus tard. Vous devez sélectionner ou créer un client pour enregistrer la dette. La vente est enregistrée et la dette est ajoutée au compte du client.

## Finaliser la vente

Après avoir sélectionné le mode de paiement, appuyez sur **Confirmer** ou **Encaisser**. La vente est enregistrée. Le stock est automatiquement mis à jour.

Une fenêtre s'affiche pour proposer d'imprimer ou d'envoyer le reçu au client.

## Reçu client

Vous pouvez :
- Imprimer le reçu sur une imprimante thermique Bluetooth.
- Envoyer le reçu par SMS ou WhatsApp au client.
- Ignorer si le client n'a pas besoin de reçu.

## Annuler une vente

Si vous voulez annuler avant de finaliser, appuyez sur **Vider le panier** ou sur la flèche retour. La transaction n'est pas enregistrée.
