# 如何编辑商品信息
feature_id: edit_product
language: zh

## 编辑现有商品

如果商品的价格、库存或其他信息发生变化，您可以随时在库存模块中进行编辑。

### 进入编辑界面

1. 打开应用，点击底部导航栏中的"库存"选项。
2. 在库存列表中找到您要修改的商品。
3. **直接点击该商品所在的行**，即可进入商品详情和编辑页面。

### 可编辑的字段

进入编辑界面后，您可以修改以下信息：

**商品名称**
修改商品的显示名称。修改后，POS 界面和报表中都会显示新名称。

**售价**
调整商品的销售价格。价格调整会立即生效，影响后续所有销售。

**成本价**
更新从供应商处购入的单价。这影响利润计算，不影响销售流程。

**库存数量**
直接修改库存数量。适用于盘点后的库存调整。

**商品条码**
添加或更换商品的条形码。点击条码输入框旁边的扫描图标进行扫描。

**商品分类**
更改商品所属的分类。

**商品图片**
更换商品图片，点击现有图片即可重新选择。

### 保存修改

完成编辑后，点击页面顶部或底部的"保存"按钮。

系统会提示"保存成功"，修改立即在全应用范围内生效。

### 批量价格调整

如果需要同时调整多个商品的价格，可以使用以下方法：
- 在库存列表中，长按一个商品进入批量选择模式。
- 勾选多个商品后，点击"批量操作"选择"调整价格"。

### 库存补货

当您收到新货时，建议使用编辑功能更新库存数量，而不是删除后重新添加，这样可以保留该商品的历史销售记录。

### 注意事项

- 修改售价不会影响已完成的历史交易记录。
- 如果商品正在被 POS 界面使用（购物车中已添加），修改价格后需要重新添加到购物车才会显示新价格。
- 修改库存数量时，系统不会自动生成补货记录，建议同时在账本中手动记录进货信息。

### 常见问题

**问：修改后能撤销吗？**
答：保存后无法直接撤销，请在保存前仔细核对信息。

**问：可以看到商品的修改历史吗？**
答：目前版本暂不支持修改历史查看，建议修改前记录原始数据。
