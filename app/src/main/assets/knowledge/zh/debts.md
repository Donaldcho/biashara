# 如何管理顾客欠款
feature_id: debts
language: zh

## 欠款管理功能

BiasharaAI 的欠款模块帮助您记录、跟踪和管理顾客赊账欠款，避免漏收款项。

### 进入欠款模块

点击底部导航栏中的"欠款"或"债务"图标，进入欠款列表页面。

### 欠款的来源

欠款有两种来源：

**自动生成**
在 POS 销售时，如果选择"赊账"作为支付方式，系统会自动为关联的顾客生成一笔欠款记录。

**手动添加**
在欠款列表页面，点击右下角"+"按钮，手动输入欠款信息：
- 选择欠款顾客
- 输入欠款金额
- 填写欠款原因或备注
- 设置欠款日期

### 查看欠款列表

欠款列表默认显示所有未还清的欠款，包括：
- 顾客姓名
- 欠款金额
- 欠款日期
- 欠款原因

列表顶部显示所有未收欠款的总金额汇总。

### 查看顾客欠款详情

点击某笔欠款记录，可以查看：
- 欠款的详细信息
- 该欠款关联的原始销售记录
- 历史还款记录

### 标记欠款为已还清

当顾客还款时，按以下步骤操作：

**全额还清**
1. 在欠款列表中找到对应的欠款记录。
2. 点击该记录进入详情页面。
3. 点击"标记为已还清"或"收款"按钮。
4. 确认还款金额和日期后，点击"确认"。
5. 系统自动将该欠款状态更新为"已还清"，并从欠款列表中移除。

**部分还款**
1. 进入欠款详情页面。
2. 点击"部分还款"按钮。
3. 输入本次实际还款金额。
4. 点击"确认"。
5. 系统更新剩余欠款金额，生成还款记录。

### 欠款提醒

在顾客详情页面，可以查看该顾客的欠款情况并直接拨打电话催款。

系统的 AI 代理也会定期提醒您哪些顾客有较大欠款金额，帮助您跟踪回款情况。

### 筛选和搜索

在欠款列表页面：
- 可以按顾客姓名搜索特定顾客的欠款。
- 可以按日期范围筛选欠款记录。
- 可以按欠款金额排序，优先处理大额欠款。

### 欠款报表

在"报表"模块中，可以查看：
- 某时间段内新增的欠款总额
- 某时间段内收回的欠款总额
- 当前未收欠款总额

### 注意事项

- 标记还清后，欠款记录不会彻底删除，仍可在"已还清"标签下查看历史记录。
- 如果误操作标记了还清，可以联系技术支持恢复，或手动重新添加欠款记录。
- 建议定期核查欠款列表，确保数据与实际情况一致。

### 常见问题

**问：欠款记录可以删除吗？**
答：可以。进入欠款详情，点击删除按钮，确认后删除。但建议只在确认是错误记录时才删除。

**问：如何批量导出欠款数据？**
答：在欠款列表页面，点击右上角的导出图标，可以将欠款数据导出为表格文件。
