# 如何配置蓝牙热敏打印机
feature_id: printer_setup
language: zh

## 打印机配置简介

BiasharaAI 支持通过蓝牙连接热敏打印机，完成销售后可以直接打印小票给顾客，提升业务专业度。

### 支持的打印机类型

BiasharaAI 支持市面上常见的 58mm 和 80mm 蓝牙热敏打印机，包括：
- Epson TM 系列蓝牙打印机
- SUNMI 蓝牙打印机
- 其他支持 ESC/POS 指令集的蓝牙热敏打印机

请参考您购买的打印机说明书确认是否兼容。

### 配对步骤

**第一步：开启打印机**
打开热敏打印机的电源开关，等待指示灯显示待机状态。

**第二步：开启手机蓝牙**
进入手机系统设置 → 蓝牙，确保蓝牙已开启。

**第三步：在手机系统蓝牙中配对打印机**
1. 在手机蓝牙设置页面，点击"扫描"或"搜索设备"。
2. 等待列表中出现您的打印机设备名称（通常为"Printer_XXXX"或打印机型号名）。
3. 点击打印机设备名称进行配对。
4. 如果需要输入配对码，通常为"0000"或"1234"（请参考打印机说明书）。
5. 配对成功后，打印机出现在"已配对设备"列表中。

**第四步：在 BiasharaAI 中选择打印机**
1. 打开 BiasharaAI，进入"设置"。
2. 找到"打印机设置"选项并点击。
3. 点击"选择打印机"按钮。
4. 应用会显示已配对的蓝牙设备列表。
5. 选择您的热敏打印机。
6. 点击"测试打印"验证连接是否正常。

**第五步：确认测试打印**
如果测试打印成功，打印机会输出一张测试小票，显示"BiasharaAI 打印测试成功"字样。

### 配置小票内容

在打印机设置中，您可以自定义小票上显示的信息：

**商店信息**
- 商店名称（显示在小票顶部）
- 商店地址
- 联系电话

**Logo**
上传您的商店 Logo 图片，会打印在小票顶部。建议使用黑白图片，分辨率适中。

**页脚信息**
在小票底部显示感谢语或广告语，如"感谢您的惠顾，欢迎再次光临！"

**税号**
如需在小票上显示税号，在此处输入您的税务登记号码。

### 打印小票

完成 POS 销售后：
1. 系统弹出"交易成功"提示。
2. 点击"打印小票"按钮。
3. 打印机自动输出小票。

也可以在交易记录中找到历史交易，点击"重新打印"补打小票。

### 常见问题处理

**打印机搜索不到**
- 确认打印机已开机且在蓝牙有效范围内（通常 10 米内）。
- 重新开启手机蓝牙后再次搜索。
- 检查打印机蓝牙指示灯状态，确认蓝牙功能已激活。

**打印乱码**
- 进入打印机设置，检查"字符编码"选项，尝试切换编码（如 UTF-8、GBK）。
- 参考打印机说明书确认正确的字符编码设置。

**连接不稳定**
- 保持手机与打印机距离在 5 米以内。
- 避免蓝牙信号干扰源（如 Wi-Fi 路由器、微波炉）。
- 尝试重新配对打印机。

**问：可以同时连接多台打印机吗？**
答：目前每次只能连接并使用一台打印机。如需更换打印机，在打印机设置中重新选择即可。
