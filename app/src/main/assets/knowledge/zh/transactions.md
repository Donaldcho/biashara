# 如何查看交易记录
feature_id: transactions
language: zh

## 交易记录功能

BiasharaAI 自动记录每一笔销售、退款和其他交易，您可以随时查看和筛选历史记录。

### 进入交易记录

点击底部导航栏中的"记录"或"交易"图标，进入交易记录列表。

### 交易列表概览

交易列表按时间倒序显示所有交易，每条记录显示：
- 交易时间
- 交易类型（销售、退款、欠款还款等）
- 交易金额
- 支付方式
- 关联顾客（如有）

### 查看交易详情

点击任意一条交易记录，可以查看该笔交易的完整信息：
- 商品明细（名称、数量、单价、小计）
- 支付方式和实收金额
- 找零金额（现金交易）
- 关联顾客信息
- 操作员信息
- 备注信息

### 按日期筛选

1. 点击列表顶部的日期筛选器。
2. 选择查看方式：
   - **今天**：仅显示当天的交易。
   - **本周**：显示本周所有交易。
   - **本月**：显示本月所有交易。
   - **自定义范围**：手动选择起止日期。
3. 选择后列表自动更新。

### 按交易类型筛选

点击"类型"筛选按钮，选择要查看的交易类型：
- **全部**：显示所有类型。
- **销售**：仅显示商品销售记录。
- **退款**：仅显示退款记录。
- **赊账**：仅显示赊账交易。
- **还款**：仅显示顾客还款记录。

### 按顾客筛选

1. 点击"顾客"筛选按钮。
2. 输入顾客姓名或手机号进行搜索。
3. 选择顾客后，列表仅显示该顾客的相关交易。

### 搜索交易

在列表顶部的搜索框中输入关键词，可以搜索：
- 商品名称
- 顾客名称
- 交易备注

### 退款操作

如果需要对某笔交易进行退款：
1. 找到并点击该笔交易记录进入详情。
2. 点击"退款"按钮。
3. 选择退款商品和退款数量。
4. 确认退款方式（原路退回或现金退款）。
5. 点击"确认退款"完成操作。
6. 库存数量自动恢复，账目自动调整。

### 交易记录的导出

在交易记录列表页面：
1. 点击右上角的"导出"图标。
2. 选择导出格式（CSV 或 PDF）。
3. 选择日期范围。
4. 点击"导出"，文件保存到手机存储或通过分享功能发送。

### 注意事项

- 交易记录一旦生成，金额和商品明细不可直接修改，如需更正请使用退款功能。
- 所有交易数据存储在本地设备，建议定期备份。
- 交易记录会在报表模块中自动汇总，无需手动统计。

### 常见问题

**问：如何找到某天的总销售额？**
答：将日期筛选设置为目标日期，列表顶部会自动显示当天的销售汇总数据。

**问：可以打印交易记录吗？**
答：在交易详情页面，点击打印图标，如果已连接蓝牙热敏打印机，可以直接打印该笔交易的小票。
