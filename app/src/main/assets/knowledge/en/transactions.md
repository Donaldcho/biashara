# Transaction History
feature_id: transactions
language: en

The Transactions screen is a complete log of everything that has happened in your business — sales, returns, debt payments, expenses, and manual ledger entries.

## Viewing Transactions

1. Tap **Transactions** or **History** in the main menu.
2. The list shows all transactions, newest first.
3. Each row displays: date, type, customer/description, and amount.
4. Tap any transaction to see its full details including line items, payment method, and any notes.

## Filtering Transactions

Use the **Filter** button (funnel icon) to narrow down the list:

- **Date Range**: Today, Yesterday, This Week, This Month, Last Month, or Custom range (pick any start and end date).
- **Type**: Sales only, Expenses only, Debt Payments only, Returns only, or All.
- **Payment Method**: Cash, M-Pesa, Bank Transfer, Credit.
- **Customer**: Show all transactions linked to a specific customer.
- **Product**: Show all sales that included a specific product.

## Searching Transactions

Use the **Search** bar to find a transaction by:
- Customer name
- Product name
- M-Pesa reference number
- Receipt number

## Exporting Transactions

1. Apply any filters you want.
2. Tap **More (⋮) → Export**.
3. Choose format: **CSV** (for Excel) or **PDF** (for printing or sharing).
4. Share via WhatsApp, email, or save to your device.

## Voiding / Returning a Sale

1. Open the transaction you want to reverse.
2. Tap **Return / Void**.
3. Select the items to return (full or partial return).
4. Stock is automatically added back. The refund appears as a negative transaction.

## Tips

- Use the **This Month** filter before visiting your bank or accountant.
- M-Pesa reference numbers are searchable — useful for resolving payment disputes.
